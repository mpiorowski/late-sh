# NES ROM Assets

Bundled ROMs are legal/free homebrew or demo ROMs used by the Arcade NES Cabinet.
Do not add commercial console ROMs here.

| File | Game | Source |
| --- | --- | --- |
| `2048.nes` | 2048 | Potatis sample ROM |
| `Nova_the_Squirrel.nes` | Nova the Squirrel | https://github.com/NovaSquirrel/NovaTheSquirrel |
| `life.nes` | Life | Potatis sample ROM |
| `from_below.nes` | From Below | https://mhughson.itch.io/from-below |
| `oopis_quest.nes` | Oopi's Quest | https://kenney.itch.io/oopis-quest |
| `squirrel_domino.nes` | Squirrel Domino | https://novasquirrel.itch.io/squirrel-domino |
| `crillion.nes` | Crillion-NES | https://kevin81.itch.io/crillion-nes |
| `bobl.nes` | Bobl | https://morphcatgames.itch.io/bobl |

Notes:
- `squirrel_domino.nes` lists zlib for provided source code on itch.io.
- `crillion.nes` links to open source code from its itch.io page.
- Some itch.io assets are name-your-own-price/free downloads without a separate
  redistribution license; re-check before packaging outside this repository.
